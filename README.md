# 📅 Professional Calendar with Holiday Markers & Range Selection

A modern, feature‑rich React calendar that lets you select date ranges, write monthly notes (auto‑saved in localStorage), and view fixed holidays with a visual dot marker. The calendar includes a smooth 3D‑like flip animation when switching months.

## ✨ Features

- **Date Range Selection** – Click two dates to select a range; shows total days.
- **Monthly Notes** – Write notes for each month, automatically saved per month.
- **Fixed Holidays** – Predefined holidays (New Year, Christmas, etc.) appear as a red dot under the date and are listed below the notes section.
- **Flip Animation** – Switching months triggers a subtle fade‑out/fade‑in effect (opacity‑based flip).
- **Today Highlight** – Current date is visually distinct.
- **Professional Color Palette** – Clean neutrals with teal accents, highly readable.
- **Responsive** – Works on desktop, tablet, and mobile.

## 🛠️ Tech Stack

- **React 18** – Functional components with hooks.
- **date‑fns** – Date manipulation and formatting.
- **LocalStorage** – Persist notes per month.
- **CSS3** – Grid layout, transitions, and absolute positioning for dot markers.
